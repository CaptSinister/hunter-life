# Hunting Life Magazine

Hunting Life Magazine is my personal Monster Hunter database project for the OldSchool Monster Hunter games This webapp is inspired by Kiranico. I love that website.

## MVP
- Game: Monster Hunter 1 (PS2)
- Goal: Kiranico-style browseable database
- Sections:
  - Monsters
  - Items
  - Quests
  - Weapons
